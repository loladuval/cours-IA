# Quelques dates clés de l'IA

<div class="cols-53">

<div class="col-left">

* 1921 : l’écrivain Karel Čapek introduit le mot « robot »
* 1943 : concept de neurones artificiels
* 1950 : invention du test de Turing
* 1956 : naissance du terme « intelligence artificielle »
* 1959 : naissance du terme « machine learning »
* 1970-1990 : IA symbolique et systèmes experts
* 1990-2000 : machine learning
* 1997 : Deep Blue bat Kasparov
* 2010 : deep learning
* 2016, AlphaGo bat Lee Sedol au go
* 2020 : modèles de langage LLM
