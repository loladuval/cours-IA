# Définitions

<dl>
<dt>Intelligence artificielle</dt>
<dd>Multiples définitions : tentative de résoudre les problèmes pour lesquels nous n’avons pas de solution algorithmique
</dd>
<dt>IA Faible</dt>
<dd>Les IAs actuelles sont performantes dans certaines tâches mais ne savent pas tout faire</dd>
<dt>IA Forte</dt>
<dd>Programme capable d’effectuer n’importe quelle tâche intellectuelle qu'un être humain peut accomplir</dd>
<dt>IA symbolique</dt>
<dd>Basée sur des règles logiques édictées par l'humain (Si "A" alors "B")</dd>
</dl>
