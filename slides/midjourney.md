# Midjourney <span class="logo midjourney-logo"> </span>

<div class="cols-53">

<div class="col-left">

* Prix : $10/month for the Basic Plan that allows you to generate ~200 images/month and provides
commercial usage rights.
* Très bonne qualité des images généreés
* The community is a great way to get inspiration
* Les images générées sont par défaut public
* Système “omni-reference” permettant de garder une cohérence graphique des personnages et des objets
dans différentes scènes
* Midjourney's help docs selecting model versions and upscaling to using character references and its
personalization tools.
